# TankAlarm 112025 Viewer - Blues Opta

Read-only monitoring device for TankAlarm system using Arduino Opta with Blues Wireless Notecard and Ethernet connectivity.

## Quick Links

- Viewer firmware: `TankAlarm-112025-Viewer-BluesOpta.ino`
- Viewer website preview: [WEBSITE_PREVIEW.md](WEBSITE_PREVIEW.md)
- System tutorials index: [../Tutorials/Tutorials-112025/README.md](../Tutorials/Tutorials-112025/README.md)
- Notecard I2C diagnostics/recovery: [../Tutorials/Tutorials-112025/I2C_UTILITY_GUIDE.md](../Tutorials/Tutorials-112025/I2C_UTILITY_GUIDE.md)

## Notes

- Use the I2C utility when Viewer serial logs indicate Notecard transport errors (for example, `{io}` failures).
- After diagnostics/recovery, re-upload `TankAlarm-112025-Viewer-BluesOpta.ino` for normal operation.
